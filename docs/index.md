## New Project Summary
Jereme Carne - P03 Data Pipeline Project
This project reads data from several different file types, processes the data, and writes a summary of the results. It was built as part of a Python data analytics course to practice working with real-world data.

One of the highlights is a custom pipeline that analyzes real flight data exported from a Dynon avionics system. It reads cylinder head temperature (CHT) readings from all four engine cylinders across an entire flight and reports the minimum, maximum, and average temperatures in Fahrenheit — the kind of analysis that could help a pilot monitor engine health over time.

The project follows a clean, organized structure where each step — reading, processing, checking, and writing — is handled separately, making the code easy to follow and build on.

## How to run
From the root project folder:
uv run python -m datafun.app_jcarne

## Insights and findings
The CHT pipeline was run against a real flight data file captured by a Dynon avionics system. The data contained over 14,000 readings across all four cylinders throughout the flight.

The analysis revealed the range of cylinder head temperatures experienced during the flight, from the coolest readings during low-power phases like taxi and descent, to the peak temperatures reached during climb. Tracking the average CHT across all four cylinders gives a useful baseline for comparing future flights and spotting any cylinder that may be running hotter than the others over time.

This project showed how Python can turn a raw avionics data export into a simple, readable engine health summary with just a few lines of code.
